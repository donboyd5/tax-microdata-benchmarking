# 2024 SIPP PUBLIC USE DATAFILE

The Survey of Income and Program Participation (SIPP) public use data
file (pipe-delimited text data) --- `pu2024_csv.zip` --- was
downloaded from this [Census Bureau
website](https://www.census.gov/programs-surveys/sipp/data/datasets/2024-data/2024.html).

After being unzipped and then gzipped, the downloaded file is roughly
100MB in size.
