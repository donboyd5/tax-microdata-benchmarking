# CEX PUBLIC USE QUARTERLY DATAFILES FOR 2023

The Consumer Expenditure Survey (CEX) public use quarterly data files
(comma-delimited text data) --- `fmli232.csv`, `fmli233.csv`,
`fmli234.csv`, and `fmli241.csv` --- were downloaded as part of the
2023 interview zip file from this [Bureau of Labor Statistics
website](https://www.bls.gov/cex/pumd_data.htm#csv).

After being gzipped, the downloaded files are of modest size.
